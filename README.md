# Tic Tac Toe

Tic Tac Toe game created in Java using Swing

## Installation

1. Clone the Repository to your www directory
   ```
  	https://github.com/hadiaamir/Tic-Tac-Toe.git
   ```
 
## How To Compile/Run

1. Compile: javac *.java

2. Run: java TicTacToeView
